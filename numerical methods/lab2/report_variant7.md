# Лабораторная работа 2: численное дифференцирование

## Вариант 7

- Функция: `f(x) = ln(1 + sin^2(x))`
- Интервал: `x in [-1.5, 1.5]`

Требуется:
- вычислить первую производную правыми и центральными разностями;
- вычислить вторую производную центральными разностями 2-го и 4-го порядка;
- оценить погрешность и исследовать зависимость ошибки от шага сетки.

## Использованные формулы

### Точные производные

- `f'(x) = sin(2x) / (1 + sin^2(x))`
- `f''(x) = [2*cos(2x)*(1 + sin^2(x)) - sin^2(2x)] / (1 + sin^2(x))^2`

### Численные схемы

- Первая производная, правая разность, `O(h)`:
  `f'(x_i) ~= (f_{i+1} - f_i) / h`
- Первая производная, центральная разность, `O(h^2)`:
  `f'(x_i) ~= (f_{i+1} - f_{i-1}) / (2h)`
- Вторая производная, центральная, `O(h^2)`:
  `f''(x_i) ~= (f_{i+1} - 2f_i + f_{i-1}) / h^2`
- Вторая производная, центральная, `O(h^4)`:
  `f''(x_i) ~= (-f_{i+2} + 16f_{i+1} - 30f_i + 16f_{i-1} - f_{i-2}) / (12h^2)`

На границах, где шаблон схемы не помещается, значения не вычисляются.

## Параметры расчета

- Основная сетка для таблиц и графиков: `n = 60`, `h = 0.05`
- Для исследования сходимости: `n = [20, 40, 80, 160, 320]`

## Результаты (максимальная ошибка)

По результатам запуска скрипта:

- `f'`, правая разность `O(h)`: `4.9896055021e-02`
- `f'`, центральная разность `O(h^2)`: `1.7842759367e-03`
- `f''`, центральная `O(h^2)`: `4.1577991760e-03`
- `f''`, центральная `O(h^4)`: `3.5131078066e-05`

## Вывод

- Полученные численные результаты соответствуют ожидаемым порядкам точности схем.
- При уменьшении шага в 2 раза ошибка для правой разности уменьшается примерно в 2 раза (`O(h)`),
  для центральных схем 2-го порядка примерно в 4 раза (`O(h^2)`),
  для схемы 4-го порядка примерно в 16 раз (`O(h^4)`).
- Наиболее точной среди реализованных оказалась центральная схема 4-го порядка для второй производной.
