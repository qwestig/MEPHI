# Лабораторная работа 2 - Расписание с ограничениями (Уровень 1)

Синтетическое недельное расписание на SWI-Prolog.

## Файлы
- `schedule.pl` - база знаний и предикаты
- `queries.pl` - 12 тестовых запросов с ожидаемыми результатами (plunit)

## Модель данных
Основной факт:

```prolog
занятие(Группа, Предмет, Преподаватель, День, Время, Аудитория).
```

Выполнен минимум для уровня 1:
- 5 групп (`g1..g5`)
- 10 предметов
- 5 преподавателей
- 10 аудиторий (`a101..a110`)
- одна учебная неделя (`mon..fri`)

## Реализованные предикаты (Уровень 1)
- `свободна_аудитория/3`
- `нагрузка_группы/2`
- `окна_в_расписании/2`

Дополнительные предикаты для анализа:
- `когда_у_группы_предмет/4`
- `группы_преподавателя/2`
- `свободные_дни_преподавателя/2`

## Запуск тестов и демонстрации вывода
Из директории `lab2`:

```bash
swipl -q -l queries.pl -g run_level1_checks -t halt
```

Команда запускает:
- unit-тесты `plunit`;
- демонстрационный вывод каждого тест-кейса в терминал: название теста, ожидаемый результат, фактический результат и статус `PASS/FAIL`.

## Интерактивный REPL
Из директории `lab2`:

```bash
swipl -q -l schedule.pl
```

Примеры запросов:

```prolog
?- когда_у_группы_предмет(g1, logic, Day, Time).
?- setof(R, свободна_аудитория(wed, '10:00', R), Rooms).
?- нагрузка_группы(g3, Count).
?- окна_в_расписании(g4, Windows).
?- свободные_дни_преподавателя(petrova, Days).
```
